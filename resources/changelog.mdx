## Overview

This page tracks all notable changes, improvements, and bug fixes to the Laravel Active Email package. We follow [Semantic Versioning](https://semver.org/) for version numbering.

<Info>
For detailed release information and to view the source code for each version, visit the [GitHub Releases page](https://github.com/veeqtoh/laravel-active-email/releases).
</Info>

## Upcoming Features

The following improvements are planned for future releases:

- **Automated Domain Updates**: Crawl the web to grab an updated list of disposable domains
- **Scheduled Updates**: Setup a schedule for automatic domain list updates

<Note>
If you have suggestions for new features or improvements, please [open an issue](https://github.com/veeqtoh/laravel-active-email/issues) on GitHub or submit a pull request.
</Note>

## Contributing to the Changelog

When contributing to this package, please ensure your pull requests include:

- A clear description of what changed
- The type of change (added, changed, deprecated, removed, fixed, security)
- Any breaking changes that require user action
