This guide will walk you through adding disposable email validation to your Laravel application. You'll have working email validation in just a few steps.

## Prerequisites

Before you begin, make sure you have:
- Installed the package via Composer (see [Installation](/installation))
- Published the configuration file (optional, but recommended)

## Add validation to your forms

Laravel Active Email provides two ways to validate emails. Choose the approach that fits your coding style.

<Tabs>
  <Tab title="Validator string">
    Add `notblacklisted` to your validation rules:

    ```php
    public function store(Request $request)
    {
        $validated = $request->validate([
            'name' => 'required|string|max:255',
            'email' => 'required|email|notblacklisted',
            'password' => 'required|min:8',
        ]);

        // Create user account
        $user = User::create($validated);

        return redirect()->route('dashboard');
    }
    ```

    <Info>The `notblacklisted` validator is automatically registered by the service provider.</Info>
  </Tab>

  <Tab title="Validation rule class">
    Import and use the `NotBlacklistedEmail` class:

    ```php
    use Veeqtoh\ActiveEmail\Rules\NotBlacklistedEmail;

    public function store(Request $request)
    {
        $validated = $request->validate([
            'name' => 'required|string|max:255',
            'email' => ['required', 'email', new NotBlacklistedEmail()],
            'password' => 'required|min:8',
        ]);

        // Create user account
        $user = User::create($validated);

        return redirect()->route('dashboard');
    }
    ```

    <Tip>The class approach provides better IDE autocomplete and type safety.</Tip>
  </Tab>
</Tabs>

## Test the validation

Let's verify that disposable emails are being blocked:

<Steps>
  <Step title="Try a disposable email">
    Submit your form with a known disposable email address like `test@mailinator.com` or `user@tempmail.com`. The validation should fail with an error message.

    ```php
    // Example test
    $response = $this->post('/register', [
        'name' => 'Test User',
        'email' => 'test@mailinator.com',
        'password' => 'password123',
    ]);

    $response->assertSessionHasErrors('email');
    ```
  </Step>

  <Step title="Try a legitimate email">
    Submit your form with a real email provider like `user@gmail.com`. The validation should pass.

    ```php
    // Example test
    $response = $this->post('/register', [
        'name' => 'Test User',
        'email' => 'user@gmail.com',
        'password' => 'password123',
    ]);

    $response->assertSessionHasNoErrors();
    ```
  </Step>

  <Step title="Customize error message (optional)">
    If you want a custom error message, update your config file:

    ```php config/active-email.php
    return [
        'error_message' => 'Please use a permanent email address.',
        // ... other config
    ];
    ```
  </Step>
</Steps>

## Use in form requests

For cleaner controllers, use Form Requests:

<Steps>
  <Step title="Create a form request">
    ```bash
    php artisan make:request RegisterRequest
    ```
  </Step>

  <Step title="Add validation rules">
    ```php app/Http/Requests/RegisterRequest.php
    <?php

    namespace App\Http\Requests;

    use Illuminate\Foundation\Http\FormRequest;
    use Veeqtoh\ActiveEmail\Rules\NotBlacklistedEmail;

    class RegisterRequest extends FormRequest
    {
        public function authorize(): bool
        {
            return true;
        }

        public function rules(): array
        {
            return [
                'name' => 'required|string|max:255',
                'email' => ['required', 'email', new NotBlacklistedEmail()],
                'password' => 'required|min:8|confirmed',
            ];
        }

        public function messages(): array
        {
            return [
                'email.notblacklisted' => 'Please use a permanent email address.',
            ];
        }
    }
    ```
  </Step>

  <Step title="Use in controller">
    ```php
    use App\Http\Requests\RegisterRequest;

    public function store(RegisterRequest $request)
    {
        // Validation already happened
        $validated = $request->validated();

        $user = User::create($validated);

        return redirect()->route('dashboard');
    }
    ```
  </Step>
</Steps>
